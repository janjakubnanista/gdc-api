create(projectData) → {Promise}

Create new project on server. Returned promise is resolved after the project is in `ENABLED` state. Passed object is a simplification of project definition. It should contain following properties: - token - driver (Defaults to `Pg`) - title - summary - template Returned promise is resolved with project data.

Invite user to this project.
